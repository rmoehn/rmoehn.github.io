Map clipboard contents

Sometimes I want to transform the contents of the clipboard before pasting them somewhere. On macOS I wrote ~/bin/pbtr for this:

#!/bin/bash

pbpaste |$* |pbcopy

For example, if you've copied some formatted text and you want to paste it as plain text, run pbtr cat. ReflowMarkdown is another application.
